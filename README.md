# 🏢 Enterprise E-Commerce Platform (Full Version)

## 🚀 Overview
A production-ready enterprise backend system built using Java and Spring Boot.

## ✨ Features
- Product Management APIs
- Order Processing
- Payment Integration (Factory Pattern)
- Discount Engine (Strategy Pattern)
- Notifications (Observer Pattern)

## 🧰 Tech Stack
Java, Spring Boot, MySQL, JPA, Maven, Docker

## ⚙️ Setup
```bash
mvn clean install
mvn spring-boot:run
```
